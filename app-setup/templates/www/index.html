Installation Instructions
  • macOS: Double-click the downloaded file, add to Keychain, mark as "Always Trust"
  • Windows: Right-click → "Install Certificate" → "Local Machine" → "Trusted Root Certification Authorities"
  • iOS: Tap the downloaded file, then go to Settings → General → VPN & Device Management to install the profile, then enable full trust in Settings → General → About → Certificate Trust Settings
  • Android: Settings → Security → Install certificates

Restart your browser after installation to eliminate security warnings.
